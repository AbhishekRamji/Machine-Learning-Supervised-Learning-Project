### Broader Cohort Groups
#### Based on the tenure column values, creating a new column called Tenure Cohort that creates 4 separate categories:**
   * '0-12 Months'
   * '12-24 Months'
   * '24-48 Months'
   * 'Over 48 Months'    


```python
# CODE HERE
def cohort(x):
    if x<13:
        return '0-12 months'
    elif x>12 and x<25:
        return '12-24 months'
    elif x>24 and x<49:
        return '24-48 months'
    elif x>48:
        return 'Over 48 months'

df['Tenure Cohort'] = df['tenure'].apply(cohort)
```
